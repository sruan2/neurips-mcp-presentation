# MCP Explorer Presentation

**NeurIPS 2025 Educational Resource Showcase**

An elegant, interactive presentation showcasing the MCP Explorer learning experience - a narrative-driven web application that teaches high school students about Anthropic's Model Context Protocol through hands-on experimentation.

## About This Presentation

This presentation highlights:

- **Project Overview**: How MCP Explorer makes advanced AI concepts accessible to high school students
- **Pedagogical Approach**: Narrative-driven, constructivist learning design with scaffolded progression
- **Interactive Features**: Side-by-side comparisons, live AI chat, and achievement system
- **Educational Impact**: Democratizing AI education for ages 14-18 with no prerequisites
- **Technical Implementation**: Modern web stack (React 18, Tailwind CSS, Framer Motion, Claude API)

## Live Demo

**Try the MCP Explorer learning experience**: [https://mcp-explorer.vercel.app](https://mcp-explorer.vercel.app)

## Run Presentation Locally

**Prerequisites:** Node.js 16+

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   - Visit [http://localhost:5173](http://localhost:5173)
   - The presentation will automatically reload when you make changes

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `dist` folder.

## About MCP Explorer

MCP Explorer is a 10-15 minute interactive learning module designed for high school students (ages 14-18) to understand:

- What is Model Context Protocol (MCP)
- The difference between AI WITH and WITHOUT MCP
- How MCP servers provide specialized capabilities
- Real-world applications relevant to student life

### Learning Objectives

- **Understand** MCP as a bridge between AI and external tools
- **Compare** AI capabilities with different MCP configurations
- **Evaluate** practical benefits through hands-on experimentation
- **Apply** knowledge to identify real-world use cases

## Technology Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: Tailwind CSS for responsive design
- **Animations**: Framer Motion for smooth interactions
- **3D Graphics**: Three.js with React Three Fiber
- **Icons**: Lucide React
- **Build Tool**: Vite
